ORDER

CRUD כתובת פרמטרים Body מה עושה איזה סוגי שגיאות יכולות לחזור
GET /api/orders אין אין שליפת כל ההזמנות 400 - בעיית שרת, לא ניתן לשלוף את ההזמנות
POST /api/orders body: userId, address, products נתוני הזמנה במבנה JSON (כולל כתובת, מוצרים, userId) הוספת הזמנה חדשה 404 - אם לא נמצא משתמש עם userId, 400 - אם לא כל הפרמטרים נמסרו
DELETE /api/orders/:id id אין מחיקת הזמנה על פי מזהה, אם היא לא יצאה לדרך 404 - לא נמצא הזמנה עם ה-ID הזה, 400 - לא ניתן למחוק הזמנה שנשלחה
GET /api/orders/:id userId אין שליפת כל ההזמנות של משתמש מסוים 404 - לא נמצאו הזמנות עם ה-userId הזה
PUT /api/orders/:id id אין (מעודכן בשרת) עדכון סטטוס ההזמנה ל-"נשלחה" 404 - לא נמצא הזמנה עם ה-ID הזה

PRODUCT

CRUD כתובת פרמטרים Body מה עושה איזה סוגי שגיאות יכולות לחזור
GET /api/products אין אין שליפת כל המוצרים 400 - בעיית שרת, לא ניתן לשלוף את המוצרים
GET /api/products/:id id אין שליפת מוצר על פי מזהה 404 - לא נמצא מוצר עם ה-ID הזה
POST /api/products body: name, price שם המוצר ומחירו במבנה JSON הוספת מוצר חדש 404 - אם לא נמסרו שם או מחיר, 400 - בעיית שרת בהוספת המוצר
DELETE /api/products/:id id אין מחיקת מוצר על פי מזהה 404 - לא נמצא מוצר עם ה-ID הזה
PUT /api/products/:id id שם המוצר ומחירו במבנה JSON עדכון מוצר על פי מזהה 404 - לא נמצא מוצר עם ה-ID הזה, 404 - אם לא נמסרו שם או מחיר

USER

CRUD כתובת פרמטרים Body מה עושה איזה סוגי שגיאות יכולות לחזור
GET /api/users אין אין שליפת כל המשתמשים (ללא סיסמאות) 400 - בעיית שרת בשל שליפת המשתמשים
GET /api/users/:id id אין שליפת משתמש לפי מזהה (ללא סיסמאות) 404 - לא נמצא משתמש עם ה-ID הזה
POST /api/users body: email, password שם משתמש, סיסמא במבנה JSON הוספת משתמש חדש 400 - אם לא נמסרו פרטי סיסמא או דוא"ל
PUT /api/users/:id id שם המשתמש/מייל/שדות אחרים במבנה JSON (לא סיסמא) עדכון פרטי משתמש (ללא סיסמא) 404 - לא ניתן לעדכן את הסיסמא, 404 - לא נמצא משתמש עם ה-ID הזה
PUT /api/users/password/:id/ id password במבנה JSON עדכון סיסמא של משתמש 404 - לא נמצא משתמש עם ה-ID הזה, 404 - ניסה לעדכן פרטים אחרים כמו שם או דוא"ל
POST /api/users/:id body: userName, password שם משתמש וסיסמא במבנה JSON כניסת משתמש עם שם וסיסמא 404 - שם המשתמש או הסיסמא לא נכונים
